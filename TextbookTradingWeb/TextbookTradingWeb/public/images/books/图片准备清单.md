# 教材图片准备清单

## 需要准备的图片数量

根据测试数据，你需要准备以下图片：

### 1. 教材封面图片（100张）
- 文件名：`covers/book_1.jpg` 到 `covers/book_100.jpg`
- 尺寸：200x300 像素
- 格式：JPG 或 PNG

### 2. 教材详情图片（60张）
前20本教材各2-4张详情图：
- `details/book_1_1.jpg`, `details/book_1_2.jpg`, `details/book_1_3.jpg`
- `details/book_2_1.jpg`, `details/book_2_2.jpg`, ...
- 尺寸：400x600 像素

### 3. 用户头像（50张）
- 文件名：`avatars/avatar_1.jpg` 到 `avatars/avatar_50.jpg`
- 尺寸：150x150 像素
- 格式：JPG 或 PNG

### 4. 轮播图（10张）
- 文件名：`banners/banner_1.jpg` 到 `banners/banner_10.jpg`
- 尺寸：1200x400 像素
- 格式：JPG

## 快速开始方案

### 方案A：最小化准备（用于快速演示）
只准备以下图片即可：
1. **默认封面**：`covers/default.jpg` - 一张通用的书本图片
2. **默认头像**：`avatars/default.jpg` - 一张通用的用户头像
3. **默认轮播图**：`banners/default.jpg` - 一张宣传图

系统会自动使用默认图片替代缺失的图片。

### 方案B：部分准备（用于较好演示）
准备10-20本热门教材的封面：
1. 《数据结构与算法分析》- `covers/book_1.jpg`
2. 《深入理解计算机系统》- `covers/book_2.jpg`
3. 《Java核心技术》- `covers/book_3.jpg`
4. 《计算机网络》- `covers/book_5.jpg`
5. 《Python编程》- `covers/book_6.jpg`
... 等等

### 方案C：完整准备（用于完美演示）
准备所有100本教材的封面图片。

## 图片获取建议

1. **豆瓣读书**：搜索书名，右键保存封面图
2. **京东/当当**：搜索书名，下载商品图片
3. **百度图片**：搜索"书名 + 封面"
4. **自己拍摄**：如果有实体书，直接拍照

## 临时方案

在准备图片之前，系统会继续使用在线占位图片服务（Picsum Photos），不影响功能测试。

等你准备好图片后，只需：
1. 将图片放入对应目录
2. 执行更新后的SQL文件
3. 刷新页面即可看到本地图片
